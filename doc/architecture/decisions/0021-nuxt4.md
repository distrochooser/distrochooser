# 21. Nuxt4

Date: 2026-03-20

## Status

Accepted

## Context

There is a newer Nuxt version (v4)

## Decision

The app is updated to v4.

The folder structure is kept at the v3 version:


We will use the old v3 structure for now, which is possible. See https://nuxt.com/docs/4.x/getting-started/upgrade#new-directory-structure
> Nuxt now defaults to a new directory structure, with backwards compatibility (so if Nuxt detects you are using the old structure, such as with a top-level app/pages/ directory, this new structure will not apply).

## Consequences

Further changes will be applied if needed.
